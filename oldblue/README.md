## Quick Start

1. Clone and Install:
```bash
git clone https://github.com/rsayeid/metering-management-analysis.git
cd metering-management-analysis
npm install
```

2. Configure Amplify:
```bash
npm run amplify:configure
```

3. Start Development:
```bash
# Start Amplify sandbox
npm run sandbox

# In a new terminal, start frontend
npm run frontend:dev
```
